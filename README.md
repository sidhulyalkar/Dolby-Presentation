# Dolby Interview Presentation - Slidev

This is a sli.dev presentation about Modular and Scalable Neural Data Pipelines.

## Setup

1. Install dependencies:
```bash
npm install
```

## Usage

### Development Mode
Run the presentation in development mode with live reload:
```bash
npm run dev
```

Then open your browser to http://localhost:3030

### Build for Production
Build the presentation for production:
```bash
npm run build
```

The built files will be in the `dist` folder.

### Export to PDF
Export the presentation to PDF:
```bash
npm run export
```

## Features

- 24 slides covering neural data pipelines
- Interactive diagrams and code examples
- Speaker notes for each slide
- Keyboard navigation (arrow keys, space, etc.)
- Beautiful gradient backgrounds
- Syntax highlighting for code blocks

## Navigation

- **Arrow keys / Space**: Next/Previous slide
- **N**: Toggle speaker notes (presenter mode)
- **O**: Toggle slides overview
- **D**: Toggle dark mode
- **F**: Toggle fullscreen
- **G**: Go to specific slide

## Presenter Mode

Press **N** to open presenter mode in a new window. This shows:
- Current slide
- Next slide preview
- Speaker notes
- Timer

## Customization

Edit `slides.md` to modify the presentation content. The theme and styling can be customized through the frontmatter configuration at the top of slides.md.
